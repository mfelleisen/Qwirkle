## Qwirkle 

The game is called Q. The package is called [Qwirkle](https://en.m.wikipedia.org/wiki/Qwirkle/).

### Install 

```
$ git clone git:...
$ cd Qwirkle
$ raco pkg install
$ raco doc Qwirkle 
```

This installs the package and opens the game description in the default browser.
